# 2′-Modified RNA Duplex Stability Calculator

Static GitHub Pages website for nearest-neighbor prediction of RNA/RNA, RNA/DNA, 2′-OMe RNA/RNA, 2′-F RNA/RNA, 2′-MOE RNA/RNA, and LNA RNA/RNA duplexes under non-crowding and cell-like conditions.

## Deployment
Upload all files in this folder directly to the repository root and publish GitHub Pages from `main` / `/(root)`.

## Input notation
- RNA: A, C, G, U
- DNA: dA, dC, dG, dT
- 2′-OMe: Ao, Co, Go, Uo
- 2′-F: Af, Cf, Gf, Uf
- 2′-MOE: Am, Cm, Gm, Um
- LNA: Al, Cl, Gl, Ul

The default 2′-OMe example is calculated automatically when the page opens.
